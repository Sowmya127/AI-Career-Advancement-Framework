# AI Career Advancement Framework

An AI-powered career development platform designed to help users improve their job readiness through mock interviews, resume analysis, career roadmap generation, and skill assessment quizzes.

## Features

* AI-Powered Mock Interviews
* Resume Analysis
* Resume-to-Job Matching
* Career Roadmap Generation
* Skill Assessment Quizzes
* Interview History Management
* User Authentication and Profile Management

## Technologies Used

* Next.js
* React
* TypeScript
* Supabase
* Tailwind CSS
* Retell AI
* OpenAI APIs

## Modules

### Mock Interview Module

Conduct AI-powered interview sessions and receive interview-related feedback.

### Resume Analysis Module

Upload and analyze resumes to identify strengths and improvement areas.

### Resume-to-Job Matching

Compare resumes against job descriptions and evaluate compatibility.

### Career Roadmap Generator

Generate personalized career development roadmaps based on user goals.

### Skill Assessment Quiz

Assess technical and professional skills through AI-generated quizzes.

## Setup Procedure

1. Clone the repository.
2. Install dependencies using npm or yarn.
3. Configure environment variables.
4. Start the development server.
5. Connect a Supabase project for database functionality.

## Note

The Supabase project used during development is currently inactive. Some database-related features may require connecting a new Supabase instance.
